# What is FTPManager ?

This is just a simple proof of concept of how FTP can be implemented in C#.

# What are FTPManager's features ?

- [+] Create a directory
- [+] List a directory 
- [+] Delete a directory
- [+] Modularity with the MethodConnection() function
